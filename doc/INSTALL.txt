Install:
1. XXX repo checkout
2. Toltsd ki a config/database.yml fajlt a config/database.yml.example mintajara
3. Hozd letre a yml fajlba irt adatbazisokat, hozzafereseket
4. Allitsd be a RAILS_ENV kornyezeti valtozot attol fuggoen, hogy szeretnel-e fejleszteni:
   $ export RAILS_ENV={development|production}
5. $ bundle                  # fuggosegek telepitese
6. $ echo "Wrisbi::Application.config.secret_key_base = '`rake secret`'" >config/initializers/secret_token.rb
                             # a cookie titkositashoz
7. $ rake db:schema:load     # adatbazis sema betoltese
8. $ rake plugins:migrate    # telepitett pluginek adatbazisa
9. $ rake user:create_root   # root felhasznalo felvetele
10. $ test $RAILS_ENV == production && bundle exec rake assets:precompile
11. $ rails s -p 3003        # alkalmazas inditasa
                             # A "3003" a port szama ahol futni fog a rails webszerver, lehet mas szabad port is
12. Nyithatod a bongeszodben: http://server:3003/

Plugin uninstall:
1. RAILS_ENV=development NAME=<plugin_name> VERSION=0 rake plugins:migrate
